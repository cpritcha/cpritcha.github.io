Dynamic Discrete Choice

Package for estimation of dynamic discrete choice models. In a dynamic discrete choice model an agent makes choices over time given their preferences, expectations of the future and biases. If the researcher observes the agent's states and actions over time and knows about the agent's expectations of the future and their biases then the agent's preferences can be estimated. Currently, the package only supports estimating random utility models with the nested fixed point algorithm. Post estimation, asymptotic confidence intervals are also supported.

PropertyCheck

A Julia Quickcheck/Hypothesis clone for property based testing.

In property based testing a general property about a system is tested. For instance, suppose an procedure creates a probability distribution. In order for the probability distribution to be valid no probilities should be negative and the probabilities should sum to one. Property testing would check these properties by creating random probability distributions and checking to see if both properties hold, as opposed to traditional testing where test cases would be generated by hand.
